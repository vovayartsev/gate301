Gate301

Hosted redirect from TLD.xyz to www.TLD.xyz

Download this project as a .zip file Download this project as a tar.gz file

Why

With cloud hosting (e.g. Heroku or Amazon S3), your website has no static IP which you could assign to the domain you own. Gate301.org gives you a hosted HTTP301 redirect which simply adds "www." to the original address.

Usage

In your domain zone file

@  IN  A  54.225.203.217

Now any HTTP request to your TLD.com will be redirected to www.TLD.com, which is likely a CNAME for your cloud-hosted website.

FAQ

Does it work with SSL?

NO, it doesn't

How stable is it?

I might turn down the server for a couple minutes a month for maintenance, so please don't use Gate31.io for critical production apps.

What if the IP 54.225.203.217 changes?

It's a static IP which should never change. Nevertheless, if this happens, I'll let you know through a newsletter.

How else can I set up a redirect?

  • If your domain is hosted at GoDaddy, take a look at http://support.godaddy.com/help/article/5120/redirect-urls-with-your-hosting-account
