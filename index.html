Gate 301

Redirects your-domain.com to www.your-domain.com

About

Gate301.org is a hosted HTTP-301 redirect which simply adds "www" prefix to the original URL.

Usage

In your domain zone file

@    A    198.199.105.209

Remember to subscribe to the newsletter so that I could warn you of IP address changes if they ever happen.

FAQ

What is the price?

The service is completely free. I've built it for my own needs, and then decided to make it public.

How reliable is it?

I will turn down the server for maintenance for a couple minutes a month, which means 99.9% uptime

What if the IP 198.199.105.209 changes?

It's a static IP which normally never changes. Nevertheless, if this happens, I'll let you know through a newsletter 1 month beforehand.

Does it work with SSL?

NO, it doesn't
