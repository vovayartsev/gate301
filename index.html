Gate301

Hosted redirect from TLD.xyz to www.TLD.xyz

About

Gate301.org is a hosted HTTP-301 redirect which simply adds "www" prefix to the original URL.

Usage

In your TLD.xyz domain zone file

@ IN A 54.225.203.217

Remember to subscribe to the emergency newsletter so that I could notify of any technical changes.

FAQ

Does it work with SSL?

NO, it doesn't

How stable is it?

I might turn down the server for a couple minutes a month for maintenance, so please don't use gate301.org for critical production apps.

What if the IP 54.225.203.217 changes?

It's a static IP which should never change. Nevertheless, if this happens, I'll let you know through a newsletter.

How else can I set up a redirect?

  • If your domain is hosted at GoDaddy, take a look at http://support.godaddy.com/help/article/5120/redirect-urls-with-your-hosting-account
