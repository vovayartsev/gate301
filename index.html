Gate301.io

About

Gate301.io is a hosted HTTP301 redirect which simply adds "www." to the original address.

Usage

  1. In your zone file, assign 54.225.203.217 as an IP for your TLD record.
@  IN  A  54.225.203.217
  1. Subscribe to a newsletter - this way I'll be able to let you know if any technical changes occur.

Usage Examples

You would use Gate301.io if you own example.org domain, and you want the visitors of http://example.com/ to be redirected to http://www.example.com/

FAQ

Does it work with SSL?

NO, it doesn't

How stable is it?

I might turn down the server for a couple minutes a month for maintenance, so please don't use Gate31.io for critical production apps.
